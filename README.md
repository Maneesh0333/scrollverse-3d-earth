# 🌍 TerraMotion

A cinematic, scroll-driven **3D Earth experience website** featuring immersive camera transitions, pinned storytelling sections, and smooth planetary animation powered by WebGL.


## 📌 Description

This project is an **interactive futuristic landing page** that showcases a living digital Earth through cinematic motion design and scroll-based storytelling.

The website combines **Three.js-powered 3D rendering**, **GSAP ScrollTrigger animations**, and **React component architecture** to create a premium sci-fi experience inspired by modern AI, space-tech, and next-generation product launches.


## 🚀 Features

- Real-time rotating 3D Earth using React Three Fiber
- Scroll-driven cinematic camera transitions
- GSAP ScrollTrigger pinned section storytelling
- Smooth blur/fade hero transitions
- Responsive immersive WebGL experience
- Minimal futuristic UI with layered depth effects

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- Tailwind CSS
- GSAP
  - ScrollTrigger
  - @gsap/react
- Three.js
- React Three Fiber
- React Three Drei
